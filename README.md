# Claude LeetCode Helper Skill

A production-ready Claude Skill that transforms LeetCode problems into interview-ready solutions with optimized algorithms, clean C++ implementations, dry runs, complexity analysis, edge cases, and detailed explanations.

## Features
- Optimized DSA solutions
- Clean LeetCode-ready C++ (default)
- Dry run with example input
- Time & Space Complexity
- Edge cases
- Line-by-line explanation
- Alternative approaches when relevant


## Installation
1. Download or clone this repository.
2. Copy the ZIP into your Claude Skills directory.
3. Restart Claude.
4. The skill will be available automatically.
